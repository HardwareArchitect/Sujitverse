# sujitverse

Self-hosted file sharing for family and friends. Web + Android + SFTP, all backed by a Raspberry Pi 5.

## Layout

- `api/` — FastAPI backend (Python)
- `web/` — React + Vite + TypeScript frontend
- `android/` — Kotlin app (planned)
- `deploy/` — deployment scripts

## Live

- Web: https://files.sujitverse.io
- API: https://files.sujitverse.io/api/

See `api/README.md` and `web/README.md` for per-component setup.
